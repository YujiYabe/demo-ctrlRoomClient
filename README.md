
# ctrlRoomClient

## 概要
エアコン＆照明リモコンアプリ


## 機能一覧
- スマホからfirebaseのリアルタイムDBを操作
- raspberry piでfirebaseの値が変更されたら対象の機器を操作


## 使用技術
ハードウェア
- raspberry pi zero
- [raspberry pi zero 赤外線拡張キット](https://bit-trade-one.co.jp/adrszirs/)	

frontend
- firebase real time data base
- nuxt.js (firebase hosting)


backend
- node.js
